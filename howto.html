How to create a page just like this one on GitHub

by Ilya Flaks, student

May 24th, 2022

First of all, you are going to need a GitHub profile. Head over to github.com and click Sign Up in the upper right corner. Enter your e-mail, the preferred user name and a password. Confirm your e-mail and sign in to your GitHub account.

On your computer, create an index.html file using VSCode or the command line or even a text editor. Edit this file and write the following in it:


<html>

<h1> ________'s (insert your name) portfolio < ⁄ h1 >

<img src=./myphoto.jpg ⁄ >

< ⁄ html >


Pick a picture of yourself and copy it to the same folder where index.html is stored. You can name the file anything you want, but make sure to edit the code above after the <img src= to reference that file name instead of myphoto.jpg

On the GitHub website, click on your avatar in the upper right hand corner to open a drop down menu, go to "Your Repositories". Press the green button that says "New"

Under "Repository Name" type a name you want. Could be your first and last names in lower case letters with no spaces. Follow it with .github.io. On the bottom of the page, click "Create Repository".

First thing you'll see is going to be the quick set up page. Click "upload an existing file". That will take you to a page where you can drag and drop your index.html and your photo files.

You will see a list of the files you drag and dropped and a green button "Commit changes". Click that button!

Now your Repository is ready to access! You can type yourfirstandlastname.github.io (for example) in the URL line in your browser and you'll see your page!

If you don't see your page, go to <Code> in the upper left corner. On the right, under Environments, click "github-pages", that should take you to your newly deployed page
